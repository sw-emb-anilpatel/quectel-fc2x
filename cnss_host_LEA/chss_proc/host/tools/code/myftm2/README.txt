Steps to build the myftm Tool
1) Make sure libnl and zlibs are installed and set the path in Makefile
2) Change toolchain path in Makefile and setup build environment
   (for QCN7606.LEA.1.1.C1 just run "source /opt/fsl-imx-xwayland/4.14-sumo/environment-setup-aarch64-poky-linux" for this step)
3) make
